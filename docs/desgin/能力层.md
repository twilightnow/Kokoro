# 能力层

所属项目：[[Kokoro]]

---

## 技术选型

| 功能 | 方案 | 成本估算 |
|------|------|--------|
| 主对话 | claude-haiku-4-5-20251001 | ~$0.25/1M tokens |
| 视觉理解 | Gemini Flash 2.0 | ~$0.075/1M tokens |
| TTS（免费） | Edge TTS | 免费 |
| TTS（高质量） | GPT-SoVITS（本地） | 一次性部署 |
| Embedding | text-embedding-3-small | ~$0.02/1M tokens |

能力层只提供基础能力，不负责决定“什么时候调用”。调用策略由人格层和感知层共同约束。

---

## 月成本估算（个人使用）

- 对话：每天50轮 × 3000 tokens = 约$1.1/月
- 视觉：每天30次触发 × 1000 tokens = 约$0.07/月
- Embedding：台词库一次性建立，之后增量更新，可忽略

**总计：约$2-3/月**

成本控制原则：

- 默认每轮只走低成本主对话模型
- 只有基础感知无法判断时，才调用更重的视觉能力
- 记忆、RAG、感知结果不能无节制增加 prompt 长度

---

## GPT-SoVITS声音克隆

网上有明日香的语音素材，GPT-SoVITS可以用少量音频克隆声音，本地推理，一次部署后TTS完全免费。需要RTX显卡，CPU也能跑但较慢。

TTS 属于表现层增强，不应阻塞核心对话闭环。
